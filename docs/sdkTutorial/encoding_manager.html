Encoding manager

Encoding manager is a class that is made to help you with creating videos from OAK cameras.

Getting started

Same as Preview manager, Encoding manager works hand in hand with the Pipeline manager, so before you can use the encoder, you will first have to declare and initialize the Pipeline manager.

 1from pathlib import Path
 2from depthai_sdk import EncodingManager
 3from depthai_sdk.managers import PipelineManager
 4import depthai as dai
 5
 6
 7# Before declaring the pipeline and encoder, we must create a dictionary of streams, with fps ans values
 8encodeConfig = dict()
 9encodeConfig["color"] = 30
10
11# create encoder with above declared dictionary and path to save the file ("" will save it next to the program file)
12em = EncodingManager(encodeConfig, Path(""))
13
14# create pipeline with above mentioned streams
15pm = PipelineManager()
16pm.createColorCam(xoutVideo=True, previewSize=(300, 300))
17
18# create encoders for all streams that were initialized
19em.createEncoders(pm)
20
21# start device
22with dai.Device(pm.pipeline) as device:
23    # create stream queues
24    em.createDefaultQueues(device)
25
26    # save frames to .h files
27    em.parseQueues()

The most important part, when using the Encoding manager is that the encoder must be initialized before the pipeline. So to begin, first we create a dictionary, which will contain all streams from the OAK camera as keys, and number of fps as their values. After that we declare the Encoding manager with the dictionary, which we just declared, and the wanted path, where the files will be stored. If we send in Path(""), the files will be stored next to the program.

As you can also see after we declare the pipeline and initialize it’s sources, we must set xoutVideo to True instead of xout. And after connecting to the device we parse through the queues and save frames to files.

To save more than just the first frame, we must add the queue parser in a loop.

 1from pathlib import Path
 2from depthai_sdk import EncodingManager
 3from depthai_sdk.managers import PipelineManager
 4import depthai as dai
 5
 6
 7# first create a dictionary with wanted streams as keys and fps number with their values
 8encodeConfig = dict()
 9encodeConfig["color"] = 30
10
11# create encoder with above declared dictionary and path to save the file ("" will save it next to the program file)
12em = EncodingManager(encodeConfig, Path(""))
13
14# create pipeline with above mentioned streams
15pm = PipelineManager()
16pm.createColorCam(xoutVideo=True, previewSize=(300, 300))
17
18# create encoders for all streams that were initialized
19em.createEncoders(pm)
20
21# start device
22with dai.Device(pm.pipeline) as device:
23    # create stream queues
24    em.createDefaultQueues(device)
25
26    while True:
27        # save frames to .h files
28        em.parseQueues()
